Run each of the following commands, {\bfseries{after opening the project (file \texorpdfstring{$>$}{>} open folder) in VSCode}}, with the command pallete (Ctrl + Shift + P)\+:

{\ttfamily CMake\+: Select a Kit}\+: scans your computer for compilers and lets you select one for use. Simply click "{}unspecified"{} and let it choose for you.

{\ttfamily CMake\+: Select variant}\+: lets you select the kind of build to do. Simply click Debug.

{\ttfamily CMake\+: Configure}\+: Takes what you selected before, and generates the necessary files in build/.

You only need to run the above once.
